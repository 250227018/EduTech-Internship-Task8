# Task 8 - Dashboard Creation

## Tool Used
Power BI Desktop

## Dataset
Sample Superstore Dataset (Kaggle)
Link: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## Dashboard Features
- Bar Chart: Sales by Category
- Line Chart: Sales trend by Order Date
- Pie Chart: Sales by Segment
- KPI Cards: Total Sales, Total Profit, Total Orders
- Interactive Slicers: Region, Category, Order Date

## How to Use
- Open Task8_Dashboard.pbix in Power BI Desktop
- Use slicers to filter data by Region, Category, or Date
- Charts update automatically based on slicer selection

## Insights
- Technology has highest sales among all categories
- Consumer segment contributes 50%+ to total sales
- Sales peak observed in Q4 every year

## Interview Answers
**What is a slicer?**
Slicer ek interactive filter hota hai Power BI mein jo dashboard ke saare visuals ko ek saath filter karta hai. Jaise Region slicer mein "West" select karo to saare charts sirf West region ka data dikhayenge.

**Measure vs Dimension?**
Dimension = categorical data jo group karne ke kaam aata hai (Category, Region, Segment).
Measure = numerical data jo calculate hota hai (Sales, Profit, Quantity).
